COMMANDES Bluetooth

Le bouton Bluetooth est présent dans la fenêtre de données du relevé topographique , la fenêtre de croquis et la fenêtre des données d'étalonnage.

Pour le DistoX1 (Leica A3) et le Sap5 le bouton Bluetooth réinitialise la connexion Bluetooth.

Pour le DistoX2 (Leica X310) il ouvre un menu déroulant [T]

  • Réinitialiser la connexion Bluetooth
  • Activer/désactiver le laser du DistoX2
  • Prendre une visée ou une donnée d'étalonnage
  • Prendre trois visées successives identiques, c'est-à-dire une visée de cheminement (fenêtres de données ou de croquis)

Dans les fenêtres de données ou de croquis, les visées sont téléchargées immédiatement si le mode de communication de l'appareil est "continu".
Dans la fenêtre des données d'étalonnage, les données sont téléchargées immédiatement en fonction d'un paramètre.

Après avoir allumé le laser il y a une courte pause avant de prendre une visée. De même, il y a une pause entre les visées après une série de trois. Les longueurs de ces pauses sont définies dans les réglages.

Pour le Bric4, le bouton Bluetooth ouvre un menu déroulant [T] :

  • Réinitialiser la connexion Bluetooth
  • allumer/éteindre le laser
  • prendre une visée
  • lancer un mode scan (uniquement si la connexion Bric4 est configurée pour recevoir les données complètes)
  • éteindre l'appareil
  • réinitialiser le compteur de visées du Bric4 (uniquement si le relevé ne contient pas de données)
