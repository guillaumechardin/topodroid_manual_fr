TopoDroid MANUEL UTILISATEUR

marco.corvi@gmail.com


TopoDroid est une application Android pour la topographie des cavités souterraines. TopoDroid récupère les données brutes et vous aide à les organiser, et à produire un croquis sur site. Vous pouvez ensuite dessiner un habillage autour du squelette de la topographie tout en effectuant un relevé ce qui réduit ainsi la probabilité d'erreur entrant dans votre esquisse et vous aide à détecter les erreurs dans les données pendant que vous êtes encore dans la caverne. TopoDroid a de nombreuses fonctions pour vous aider à tirer le meilleur parti de votre travail sous terre. De cette manière, la topographie est presque terminée à la fin de votre sortie topo.

TopoDroid rend plus efficace le temps que vous passez à topographier sous terre. Il est conçu pour gérer un seul relevé à la fois. Il ne gère pas les projets de topographie des grottes complexes qui comprennent un certain nombre de séances topo différentes. C'est parce que le but de TopoDroid est d'aider le topographe à collecter les données avec plus de précision et de rendre la tâche plus agréable, et non de produire la topographie finale de la caverne. Pour ce faire, de bons programmes PC existent déjà.

TopoDroid comporte cinq "niveaux d'activité" différents. Les niveaux les plus bas disposent déjà de suffisamment de fonctionnalités pour télécharger des données, gérer les noms des stations et dessiner des croquis simples. Les niveaux supérieurs peuvent être utilisés pour préparer les données et les croquis en vue d'un travail plus facile sur le PC.

TopoDroid est conçu pour traiter un seul relevé à la fois, et non pour gérer des projets complexes d'étude de cavités comprenant plusieurs relevés différents. Néanmoins, il comprend un gestionnaire de projet qui peut être utilisé pour composer des relevés dans un projet de grotte/karst et un visualisateur 3D avec lequel vous pouvez voir la 3D des relevés et des projets topographiques.

TopoDroid fonctionne avec les appareils suivants : DistoX (v. 1, 2 et BLE), Sap5 et Bric4, mais les données des relevés peuvent être saisies manuellement.
Attention, les appareils Android (et le stylet actif) peuvent influencer les mesures prises avec un appareil électronique, s'ils sont trop proches.

Dossier privé TopoDroid

Le dossier personnel de Topodroid, "Android/data/com.topodroid.TDX/files", contient les dossiers des outils de dessin, "point", "ligne" et "zone", la base de données des périphériques "device10.sqlite" avec les données d'étalonnage, les fichiers d'exportation d'étalonnage (dossier "ccsv"), les fichiers de firmware des Distox2 et DistoX-BLE (dossier "bin") et les dump mémoire DistoX (dossier "dump").

Ce dossier et son contenu sont supprimés lorsque l'application est désinstallée. Sauf, si vous cochez la case pour conserver les données de l'application.

Dossiers de relevés TopoDroid

Le dossier de base de Topodroid est "Documents/TDX", dans le stockage externe principal. Il contient le dossier de projet "TopoDroid". Le dossier de projet contient la base de données "distox14.sqlite", le dossier "zip", le dossier "tmp" pour les fichiers temporaires et il est alimenté avec un dossier pour chaque relevé qui est créé.

Ce dossier et ses sous-dossiers ne sont pas supprimés lors de la désinstallation de l'application.


Données de relevé

Les données de relevés et les croquis sont enregistrés dans le sous-dossier "TopoDroid" de "TDX".
Chaque relevé possède un sous-dossier sous "Topodroid
. Le dossier "TopoDroid" contient la base de données ("distox14.sqlite"), les dossiers de relevés et le dossier des projets topographique ("thconfig").

Accorder les permissions

Sur Android 11 et plus, TopoDroid peut échouer à créer le dossier de base "TDX" ou d'autres sous-dossiers, parce qu'il n'a pas suffisament de permissions pour l'accès au stocakge externe.
Dans ce cas, vous pouvez accorder ces permissions en utilisant "adb" (Android Debug Bridge).
  1. Télécharger "adb" sur votre PC
  2. sur Android, activer les "options développeur" (aller dans les paramètres > Mon appareil et appuyez 7 fois sur "Build Number")
  3. Aciver le "debogage USB" dans les "Options développeur"
  4. sur le PC démarrer le serveur adb en entrant la commande : "adb start-server"
  5. connecter l'appareil Android au PC avec un cable USB. Vous aurez peut-être à autoriser "adb-connection" du PC, sur votre appareil Android
  6. sur le PC entrez les 3 commandes suivantes :
    • adb shell appops set com.topodroid.TDX READ_EXTERNAL_STORAGE allow
    • adb shell appops set com.topodroid.TDX WRITE_EXTERNAL_STORAGE allow
    • adb shell appops set com.topodroid.TDX MANAGE_EXTERNAL_STORAGE allow

Importer d'anciens relevés

TopodDroidX (v.6), peut être installé conjointement avec l'ancienne version v.5.1.40 parce qu'il utilisent différents dossiers de base.
Les anciens relevés en v.5.1.40 peuvent être importés en v.6 depuis leur fichier zip d'exportation, créés depuis l'anciene version de TopoDroid.
Ils peuvent aussi être créé avec le script zip récupérable à l'adresse suivante : http://marcocorvi.altervista.org/caving/speleoapps/speleoapks/TopoDroidZipScript.html

N.B. TopoDroid v.5.1.40 ne peut pas importer d'archives zip de la version 6 à cause de changements dans la base de données.

Caractéristiques matérielles

TopoDroid utilise les fonctionnalités matérielles suivantes: "Bluetooth", "Photographie", "Micro", "GPS", "Multitouch". Les fonctions de TopoDroid qui nécessitent une fonctionnalité non présente sur votre appareil Android sont automatiquement désactivées.
La connexion Internet est utilisée uniquement pour l'installation du manuel utilisateur et pour vérifier si une nouvelle version est disponible.

Erreurs et plantages

En raison de ressources limités, l'application ne peut pas être intégralement. Faites un backup de l'apk et de la base de données avant la mise à jour de l'application.
  • Un plantage après la sélection d'une option peut être due à une traduction incomplète. Exécutez l'application en Anglais. Si le plantage disparait, contactez l'équipe de traduction
  • Si vous découvez une erreur ou si l'application ne se comporte pas comme escompté envoyez moi un email. Indiquez la version de l'application et d'Android.
  • Pour toutes question d'ordre général ou nouvelles fonctionnalités, envoyez un email sur la liste de diffusion TopoDroid.
