GROUPES DE DONNÉES D'ÉTALONNAGE du DistoX

Cette boîte de dialogue s'ouvre en appuyant sur le bouton Groupe de la fenêtre Données d'étalonnage.

Avant de calculer le coefficient d'étalonnage les données d'étalonnage doivent être subdivisées en groupes de données ayant le même azimut et inclinaison (rotations différentes).
Attention, une mauvaise affectation de groupe peut entraîner l'échec du calcul des coefficients d'étalonnage (généralement dû à un trop grand nombre d'itérations dans l'algorithme).

Les groupes sont identifiés par un nombre entier à partir de 1.
Les données identifiées par un 0 sont répertoriées mais exclues du coefficient de calcul.
Les données supprimées ont un numéro de groupe négatif, donc définir le numéro de groupe -1 revient à marquer les données comme "Supprimées".

Il existe deux stratégies d'affectation automatique des groupes :

  1. TopoDroid : quatre données à la fois pour chaque groupe ;
  2. PocketTopo : quatre à la fois pour les 16 premières données (quatre groupes), puis chaque donnée est un groupe à part entière ;

Actions :

  • Réinitialiser tous les groupes : les groupes de données sont remis à zéro
  • OK : des groupes sont affectés (uniquement possible pour les données qui n'ont pas de groupe)
