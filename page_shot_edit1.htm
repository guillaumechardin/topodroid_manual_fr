BOITE DE DIALOGUE D'ÉDITION DES VISÉES

Cette boîte de dialogue est ouverte en appuyant brièvement sur les données d'une visée dans la fenêtre de données du relevé topographique.

Elle affiche :

  • Longueur, azimut, inclinaison. Modifiables pour les visées entrées manuellement mais pas pour les visées téléchargées.
  • Différences d'accélération, de champ magnétique et de l'inclinaison magnétique calculées respectivement à partir des valeurs moyennes. Fonctionne seulement pour les visées faites avec un DistoX2.
  • Stations
  • Flèche gauche : afficher la visée précédente
  • Bouton "inverse" : permuter les stations ("Depuis" / "Vers")
  • Flèche droite : afficher la visée suivante
  • Ligne de commentaire (note)
  • Profil d'extension (direction de la visée) avec au choix: "Gauche", "Vertical", "Droite". L'extension est "désactivée" si rien n'est coché.

Boutons :

  • visée "dupliquée" [A]
  • visée de "surface" [A]
  • visée "commentée" [A]
  • la visée est liée à la station précédente.
  • la visée est liée à la station suivante.
  • la visée est une visée arrière.
  • renuméroter les visées à partir de ce point en suivant la stratégie actuelle de dénomination des stations. Les deux stations ("DEPUIS" et "VERS") doivent être indiquées.
  • appliquer un changement de station à tout le groupe de visées d'habillage liées à une station donnée. S'applique uniquement aux visées d'habillage de la même station que la visée en cours d'édition. Seule la station "DEPUIS" doit être indiquée.
  • cacher les visées latérales soit sur le plan, soit sur la coupe, soit sur les deux (habillage seulement) [A]
  • transformer cette visée et toutes les visées de ce groupe en visées de section transverse. Les visées de section transverse sont affichées en vert. Ce dernier bouton assigne alors l'affichage des visées latérales soit au plan soit à la coupe (cette option est utile pour l'appli cSurvey).
  • Annuler : fermer la boîte de dialogue sans enregistrer
  • Enregistrer: enregistrer les modifications sans fermer la boîte de dialogue
  • Plus : passer à la boîte de dialogue d'Édition secondaire (au contenu variant suivant le niveau utilisateur choisi)
  • OK : enregistrer les modifications et fermer la boîte de dialogue
Les marqueurs de visées, "dupliquée", "surface" et "commentée", s'excluent mutuellement. Les visées marquées ne comptent pas pour les statistiques du relevé topo. En export, les visées marquées est exportées soit avec un marqueur si le format le permet, soit avec un commentaire.

Un avertissement est émis si, lors de l'édition d'une visée de cheminement, celle-ci diffère d'une autre visée de cheminement entre les mêmes stations de plus d'un quart de la longueur, ou de plus de 15°.

Copier - coller un nom de station

Les noms de stations peuvent être «coupés», «copiés» et «collés» dans l'application, à partir de n'importe quelle boîte de dialogue où se trouve un champ d'entrée de station.

Un appui long sur le champ de saisie de la station ouvre un petit menu déroulant avec les trois options suivantes :

  • COUPER
  • COPIER
  • COLLER

Réaffectation des stations
La réaffectation des stations peut être initiée uniquement à partir d'une visée de cheminement. Par défaut, la réaffectation affecte toutes les visées qui suivent la visée de cheminement sélectionnée jusqu'à la fin. Si le paramètre personnalisé splay station est désactivé, les stations des visées d'habillage ne sont pas réaffectées.

La réaffectation normale ne devrait être utilisée que pour les toutes dernières visées de cheminement, dans le cas où une visée n'a pas été détecté (par exemple, seulement deux données au lieu de trois), ou détectée de manière erronée (des visées d'habillage très proches considérées comme un cheminement).

Un meilleur contrôle sur les visées qui sont réassignées est obtenu avec le mode "multi-sélection", où l'utilisateur peut choisir les visées à réassigner individuellement.


Groupe de visées latérales d'habillage
Un groupe de visées d'habillage est constitué de toutes les visées entre deux visées de cheminement. Le changement de station d'un groupe de visées d'habillage s'applique uniquement à l'ensemble contigu de visées d'habillage ayant la même station que la visée en cours d'édition. Seule la station DEPUIS doit avoir été spécifiée.
Si le paramètre personnalisé "Splay group" est défini, seuls les visées d'habillage après la visée en cours d'édition sont modifiées.


Classes de visées d'habillage
Les classes de visées d'habillage sont activées par un "paramètre personnalisé". Il existe quatre classes et quatre couleurs d'affichage :

  • normales : visées latérales d'habillage standard. Bleu
  • visées latérales des sections transverses. Vert
  • visées latérales horizontales, sur le plan. Bleu foncé
  • visées latérales verticales, sur la coupe. Violet


Le bouton Classe d'affichage permet de faire défiler ces quatre classes. La visée d'habillage est définie dans la classe choisie lorsque les données sont sauvegardées. Si le bouton "tous les visées" est coché, la classe est également attribuée à toutes les visées d'habillage "normales" précédentes contigues.

Les classes d'affichage peuvent également être définies avec le mode "multi-sélectiont". En multi-sélectiont, vous pouvez sélectionner des visées d'habillage qui ne sont pas contigües et leur attribuer à toutes une classe d'affichage.

Si vous effectuez des visées latérales d'habillage de manière cohérente, les classes peuvent être définies en éditant la visée de cheminement et en cochant le bouton de groupe des visées d'habillage. Vous devez d'abord prendre les visées d'habillage de la section transverse en commençant par +90 et en tournant en rond. Ensuite, vous prenez les visées latérales H. Enfin, vous prenez les visées latérales dans le plan vertical en commençant à nouveau par +90.
