FUSION ET DIVISION DE CROQUIS

Dans la boîte de dialogue Renommer/supprimer le croquis, vous pouvez soit renommer, soit supprimer le croquis en cours.

Si la fonction de division/fusion [G] est activée, des boutons permettent également de diviser le croquis (c'est-à-dire d'extraire une partie des dessins pour créer un nouveau croquis) et de fusionner les dessins d'un autre croquis dans le croquis actuel.


Division du croquis
Il existe également un bouton permettant de diviser le croquis. Cela crée un nouveau croquis rempli d'éléments sélectionnés dans le croquis actuel. Pour sélectionner ces éléments, tracez une bordure qui les entoure dans la fenêtre de croquis.
Vous pouvez spécifier le nom de ce nouveau croquis. Toutefois, son origine est la même que celle du croquis actuel. De même, il possède le même type de vue en coupe que le croquis actuel.
Si la fenêtre de croquis affiche un plan, la nouvelle vue en plan du croquis est remplie avec les éléments sélectionnés, et la vue en coupe est vide. Et inversement, si la fenêtre de croquis affiche une coupe.
Seuls les éléments visibles sont sélectionnés lors de la division.


Vous pouvez choisir de déplacer les éléments sélectionnés vers le nouveau croquis ou simplement de les y copier.


Après cette opération, la fenêtre de croquis affiche le nouveau croquis.


Fusion de croquis
Si le contour d'un autre croquis est affiché, vous pouvez le fusionner avec le croquis actuel.
Les éléments du contour du croquis sont copiés dans le croquis en cours.
