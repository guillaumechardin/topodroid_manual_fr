ALIAS du DistoX

Cette boîte de dialogue est ouverte en appuyant longuement sur une entrée de périphérique dans la fenêtre de Périphérique.

Elle affiche quelques informations sur le DistoX sélectionné et dispose d'un champ de texte pour rédiger ou modifier son alias.

Si le paramètre de second dispositif est activé, une case à cocher permet de définir l'appareil choisi comme second dispositif (DistoX uniquement).


OK enregistre le nouvel alias dans la base de données et ferme la boîte de dialogue. Si la case est cochée, l'appareil est alors défini comme deuxième dispositif (DistoX uniquement).
