CODES DE COULEUR de TopoDroid

Dénomination des stations

  • visée avant [par défaut] : bleu
  • visée arrière : bleu foncé
  • visée avant + visée arrière : violet
  • visée au trépied : rose
  • anomalie magnétique : rouge
  • TopoRobot : orange


Données du relevé tpographique (tableau)
Couleur de premier plan :

  • blanc : plan de la visée
  • gris : visée répétée
  • bleu : visée latérale d'habillage
  • vert : x-splay
  • bleu foncé : h-splay
  • bleu foncé : v-splay
  • orange : visée à blanc
  • violet :
  • autres : couleur attribuée par l'utilisateur

Couleur d'arrière-plan :

  • noir : normal
  • rouge : plan non fiable
  • gris : visée commentée
  • orange : visée courte


Stations du relevé topographique (tableau)
Couleur de premier plan :

  • vert : station active

Couleur d'arrière-plan :

  • noir : normal
  • vert foncé : dernières données
  • gris : multi-sélection
  • jaune foncé : DistoX visées arrière


Visées du croquis

  • blanc : normal
  • rouge : peu fiable
  • orange : incohérence avec les visées parentes


Visées latérales d'habillage du relevé topographique (tableau)

  • bleu : normale
  • vert foncé : commentée
  • vert, bleu foncé, bleu profond : classes d'échantillonnage
  • autres : couleur attribuée par l'utilisateur


Stations du relevé topographique (tableau)

  • violet : normal
  • vert : station active
  • rouge : ligne médiane cachée après la station
  • bleu : ligne médiane cachée avant la station
  • jaune : station sauvegardée


Données d'étalonnage
Couleur de premier plan :

  • bleu, orange : groupes de données
  • gris : données non utilisées

Couleur d'arrière-plan :

  • violet : données partiellement téléchargées (non utilisées)
  • rouge : erreur supérieure à 1 degré
  • vert : grande différence par rapport à la moyenne du groupe
  • marron : données saturées
  • gris : données supprimées (groupe -1)
