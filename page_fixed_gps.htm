LOCALISATION GPS par le périphérique Android

Cette boîte de dialogue s'ouvre en appuyant sur le bouton GPS de la feneêtre de Liste des Points Géolocalisés.

Pour utiliser le GPS d'Android, cette option doit être activée. Pour des raisons de sécurité une application générique comme TopoDroid ne peut pas activer le GPS. S'il n'est pas activé, TopoDroid affiche une boîte de dialogue qui vous demande si vous voulez l'activer (par l'application "Paramètres de réglages"). N'oubliez pas de le désactiver lorsque vous avez terminé pour économiser la batterie.

Cette boite de dialogue contient :

  • Nom de la station
  • Commentaire

Boutons :

  • GPS démarrage/arrêt du pointage
  • Voir le point sur la carte d'une application cartographique spécifique
  • Enregistrer le point
Lors de la recherche des satellites le bouton "GPS" devient rouge (ou vert) indiquant le nombre de satellites visibles. Lorsque suffisamment de satellites sont en vue, le bouton devient vert et les coordonnées sont affichées en dessous. Vous devez attendre que ce bouton devienne vert pour que les coordonnées soient stables et que l'erreur de localisation soit suffisamment faible. Arrêter ensuite le pointage GPS en appuyant à nouveau sur le bouton.
