EXPORTER UNE TOPOGRAPHIE FINALISÉE AU FORMAT THERION ("map" Therion)

Cette section décrit l'utilisation de Therion ou de cSurvey pour rédiger le fichier topographique final.

Therion
Si vous voulez compiler le projet Therion, vous devez fournir un fichier thconfig qui peut être aussi simple que ça :

source my_survey.th
export map -proj plan -o my_cave_p.pdf
export map -proj extended -o my_cave_s.pdf

Vous devez avoir exporté les données du relevé topographique sous forme de fichier Therion ".th", et les croquis sous forme de fichiers Therion ".th2".
Le fichier de données contient des commandes pour entrer les fichiers de croquis. Cependant, elles sont commentées et vous devez les décommenter.
Le fichier de données contient des commandes définissant un segment (scrap) pour chaque croquis. Pour les relevés simples, vous pouvez laisser ces commandes commentées.Vous pouvez également exporter les croquis sous forme de fichiers Therion ".xvi" et les retracer en utilisant xtherion.

cSurvey
Exportez le croquis en tant que fichier cSurvey ".csx".
Ouvrez le fichier croquis exporté avec cSurvey : vous obtiendrez la feuille de données du relevé remplie avec les données topographiques et les croquis correctement alignés sur la page de dessin de cSurvey.
