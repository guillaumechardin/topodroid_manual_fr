STATIONS SAUVEGARDÉES (MEMORISÉES)

Cette boîte de dialogue s'ouvre en appuyant sur l'icône de Station sauvegardée  dans la fenêtre de données du relevé topographique ou en appuyant sur le bouton Station sauvegardée de la boîte de dialogue d'édition de station (accès par ) de la fenêtre de croquis.

Les "Stations sauvegardées ou mémorisées" sont des stations conservées pour y revenir ultérieurement : par exemple, à un carrefour, pour revenir en arrière et poursuivre le relevé topo à partir de là; ou pour marquer une suite possible destinée à une séance de topo future.

La boîte de dialogue affiche la liste des stations mémorisées. Vous en sélectionnez une en appuyant brièvement dessus.

La station sélectionnée est affichée dans les champs de texte. Ouvert à partir de la fenêtre des données du relevé, le champ de texte contient la dernière station, ou la station active si elle est définie. Ouvert à partir de la boîte de dialogue d'édition de station, le champ de texte contient la station en cours d'édition.

Chaque station enregistrée doit avoir une brève description (commentaire) ou un ensemble d'icônes indicateurs (flag).

  • station sélectionnée
  • icône indicateur de la station, le cas échéant ("fixé" ou "dessiné")
  • commentaire de cette station

Boutons :
  • Effacer les champs de la station : pour réinitialiser le commentaire et changer d'icône
  • Enregistrer la station dans la base de données
  • Supprimer la station de la base de données (non récupérable)
  • Active (Courante) : définir la station comme station "active" ou station "courante"
  • Fermer la fenêtre sans modifications
