OUTILS LIGNES

Les outils "ligne" sont définis dans des fichiers texte.

Deux entrées spécifiques : "user" et "paroi", sont intégrées dans le programme. Les autres fichiers "lignes" sont téléchargés à partir du sous-répertoire symbol/line de TopoDroid. Ce répertoire est rempli avec les outils de points compilés dans l'apk de distribution. Vous pouvez supprimer les outils dont vous n'avez pas besoin, modifier ceux que vous souhaitez modifier et en ajouter de nouveaux qui ne sont pas fournis dans l'apk.


Le fichier d'outil de ligne contient des lignes spécifiant la propriété de l'outil de dessin.

  • "symbol line" [obligatoire]
  • "name NAME", where NAME is the area default name [obligatoire]
  • translated names, eg, "name-es ..." [facultatif]
  • "th_name NAME", où NAME est le nom de type Therion et doit être le même que le nom de fichier Therion sans l'éventuel préfixe "u" [obligatoire]
  • "group GROUP", specifies the group this line belongs to [optionnel, default no group]
  • "color rgb alpha" (RVB et alphanumérique au format hexadécimal, par exemple '0x00ff00' est le vert, et '0xff' est complètement opaque) [RVB obligatoire, la partie alpha est facultative, si non spécifié 0xff est utilisé].
  • "width SIZE", largeur de ligne en unités du paramètre de largeur de ligne [facultatif, par défaut 1].
  • "dash PATTERN" exemple "15 5" [motif de tiret facultatif].
  • "style STYLE" où STYLE peut être "droit" ou "xN".[facultatif]
  • "effect" commence la ligne path-effect, est suivi par des commandes path puis et fermé par "endeffect"
  • "endeffect"
  • "roundtrip VALUE" [facultatif, pour SVG-roundtrip, VALUE peut être 1 (walls_shp), 2 (walls_sym), 3 (detail_shp), 4 (detail_sym) - par défaut 3].
  • "level" marqueur d'affichage [facultatif, 1 si absent].
  • "endsymbol"

Nom
Reportez-vous à la page outil points pour savoir comment spécifier les noms.

Groupe
Les types de ligne dans le même groupe peuvent être joints par "continuation".

Cela permet de spécifier que plusieurs lignes de "parois", bien qu'ayant des types différents, peuvent être réunies les unes aux autres.

Tiret
La directive "dash" spécifie que la ligne est dessinée avec un trait en pointillés. Par exemple, "dash 15 5" signifie que la ligne est dessinée avec des segments de 15 unités, séparés par des espaces de 5 unités.

Style
La directive "style" est utilisée pour dessiner des lignes en utilisant moins de points. Un "style x3" signifie par exemple que la ligne est dessinée en utilisant un point sur trois.
L'effet résultant est que la ligne est moins "courbée". Si vous voulez des segments vraiment droits, utilisez "style straight" (redresser ligne).

Effet de tracé (Effect path)
Cette option est utilisée pour définir des lignes avec un style complexe.

Niveau
L'attribut "level" prend la valeur du marqueur pour les vues activées ici.
L'indicateur est un nombre dont la somme est 1 (niveau de base), 2 (plancher), 4 (remplissages), 8 (plafond) et 16 (artefacts).
