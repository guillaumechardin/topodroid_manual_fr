INFOS sur le DistoX1 (Leica A3)

Cette boîte de dialogue s'ouvre à partir du bouton Info de la fenêtre du périphérique.

Informations sur le DistoX A3 :

  • code de l'appareil
  • unités d'angle
  • boussole et clino (on/off)
  • mode normal/étalonnage
  • mode silencieux (on/off)

La boîte de dialogue comporte un bouton qui efface la mémoire du DistoX, c'est-à-dire qu'elle efface le hot-bit qui marque les données à transmettre.
Après l'effacement, le DistoX doit être éteint et rallumé pour que l'action soit effective.
