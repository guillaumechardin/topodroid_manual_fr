GESTIONNAIRE DE PROJET TOPOGRAPHIQUE - Boîte de dialogue d'équivalence entre relevés

Cette boîte de dialogue s'ouvre en appuyant sur le bouton Équivalences de la fenêtre de projet.

La boîte de dialogue liste les équivalences entre les différents relevés topographiques du projet.


Une équivalence peut être supprimée en appuyant dessus : une boîte de dialogue de confirmation s'affiche alors.
