DESSIN 3D

Dessin 3D permet de dessiner le croquis du contour de la cavité en une seule étape. Par conséquent, l'affichage est centré autour de la visée en cours.

La fenêtre Dessin 3D possède trois modes (et trois barres de boutons différentes) :

  • Déplacer
  • Dessiner
  • Sélectionner
Comme pour la fenêtre le croquis 2D les boutons verts les plus à gauche concernent le mode en cours.
Les autres boutons pour le mode Déplacer sont :
  • Afficher surface
  • Créer surface
  • Téléchargement de données
  • Commentaires
  • Info

Mode d'affichage de la surface

Le mode d'affichage de la surface peut être :

  • Surface de la visée actuelle uniquement
  • Surfaces de la visée en cours et des visées proches
  • Toutes les surfaces
  • Aucune surface.

Pour Aucune surface vous pouvez accéder aux visées proches de l'une d'entre elles en la sélectionnant.

Pour le mode Dessiner vous pouvez placer des icônes sur la surface, ainsi que les symboles de ligne.

  • Annuler
  • Refaire
  • Palette (sélecteur d'élément)
Pour le mode sélectionner vous pouvez :
  • Sélectionner le mode
  • Sélectionner la taille
  • Élément précédent (non implémenté)
  • Élément suivant (non implémenté)
Menu déroulant
  • Exporter (formats pris en charge : Therion 3D, DXF)
  • Palette de couleurs
  • Ajustement du zoom
  • Principal : orientation du modèle 3D en utilisant le périphérique Android
  • Supprimer le modèle 3D
  • Effacer : supprimer toutes les surfaces
  • Paramètres
  • Aide
