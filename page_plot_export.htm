EXPORTATIONS DES CROQUIS

Les croquis sont stockés dans TopoDroid au format binaire et peuvent être exporté en tant que :

  • Fichier Therion ".th2
  • Fichier ".csx" de cSurvey
  • Fichier ".xml" de Tunnel
  • 2D DXF (LibreCAD)
  • SVG (Inkscape)
  • PDF
  • shapefile (QGIS)
  • Image xtherion ".xvi".

En plus de l'exportation, il existe une option permettant d'enregistrer automatiquement un croquis (dans l'un des formats d'exportation pris en charge) chaque fois qu'il est fermé.

Le fichier d'exportation cSurvey contient à la fois les croquis en plan et en coupe. Si des points de "section" ont été définis, il contient aussi les sections transverses. Les sections transverses ne sont pas exportées comme fichiers cSurvey.

Les croquis peuvent être exportées sous forme de fichiers DXF 2D. Les versions DXF prises en charge sont 9, 12 et 14.

L'exportation du fichier shapefile comprend les stations, les visées et le dessin. Tous les objets de dessin sont convertis en "linestring". Les tables d'attributs contiennent les données des objets (par exemple, le type de point, l'orientation, etc.). Les fichiers composant un export shapefile sont compressés dans une archive zip.

Quelques paramètres d'exportation, spécifiques au format sélectionné, peuvent être remplacés dans la boîte de dialogue d'exportation.

Exportation de la vue d'ensemble
Les croquis peuvent également être exportés à partir de la "Fenêtre d'aperçu" aux formats suivants :

  • Fichier Therion ".th2
  • DXF 2D (LibreCAD)
  • SVG
  • shapefile (QGIS)
  • PDF
  • fichier ".xvi" (xtherion)


L'exportation de la fenêtre d'aperçu inclut tous les croquis de la vue courante (plan ou coupe).
S'il y a des points de "section", les sections transverses sont également incluses.
