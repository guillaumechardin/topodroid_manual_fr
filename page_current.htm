STATION ACTIVE (ou COURANTE)

La station active ou station courante, est la station à partir de laquelle le relevé topographique va se poursuivre. Si vous avez défini une station active, TopoDroid continuera la numérotation des stations à partir de celle-ci. Si la station active n'est pas définie, c'est la dernière station du tableau des données de visées qui est utilisée.

Ainsi, vous pouvez commencer une nouvelle branche au milieu du relevé topographique en désignant comme station active la station où vous souhaitez la raccrocher. Sinon, vous pourrez rectifier la station de destination (visée "vers") de cette nouvelle branche, plus tard.

La station active ou courante est surlignée en vert dans la fenêtre de données du relevé topographique. Un appui long sur la station active la désélectionne (elle s'éteint). La station active peut être définie de trois façons différentes :

  • Avec un appui long sur le nom de la station dans la fenêtre de données du relevé topographique.
  • À partir de boite de dialogue des stations mémorisées (bouton "Active" ou "Courant")
  • Par l'intermédiaire de la boîte de dialogue d'Édition des stations dans de la fenêtre de croquis (esquisses).
