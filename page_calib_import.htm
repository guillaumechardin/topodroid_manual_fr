DistoX ÉTALONNAGE Import/Export

Importation d'étalonnage

Un étalonnage peut être importé à partir d'un fichier d'étalonnage CSV.

Cette boîte de dialogue s'ouvre en appuyant sur Importer dans la boîte de dialogue de Liste d'étalonnage.

Cette boîte de dialogue affiche la liste des fichiers CSV d'étalonnage.
Appuyez sur l'entrée que vous souhaitez importer.

L'importation est impossible (échec) si le fichier ne contient pas d'étalonnage, ou si l'étalonnage est déjà présent.
Il peut également échouer si l'adresse Bluetooth du dispositif inscrite dans l'étalonnage est différente de celle du dispositif actuel.

Exportation d'étalonnage CSV

Un étalonnage DistoX peut être exporté en format CSV (texte).

La boîte de dialogue d'exportation s'ouvre à partir du menu Export de la fenêtre d'information sur l'étalonnage.

Le format d'exportation d'un étalonnage CSV contient les informations d'étalonnage : nom, date, adresse Bluetooth du périphérique (DistoX), commentaire, algorithme; suivies des données d'étalonnage, et des coefficients d'étalonnage.

Chaque ligne de données contient :

  • Indexation des données
  • Valeurs G et M (six entiers)
  • Groupe
  • Azimut, pente et rotation
  • Erreur
  • Statuts
