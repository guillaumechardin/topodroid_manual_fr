APPLICATIONS AUXILIAIRES POUR TopoDroid

Lorsque vous travaillez avec TopoDroid, certaines applications complémentaires et auxiliaires peuvent vous être utiles :

  • Proj4 est une application de conversion de coordonnées (très simple), utilisée par TopoDroid pour convertir des valeurs de latitudes et longitudes en d'autres systèmes de coordonnées.
  • MobileTopographer est une application de localisation. Vous pouvez l'utiliser pour relever les coordonnées du point et demander à TopoDroid de les lire à partir du fichier de points dans la liste de points de MobileTopographer.
  • CrowdMag est une application du NOAA National Geophysical Data Center, utile pour obtenir des informations sur l'état actuel du champ magnétique terrestre.
  • Une application pour prendre des photos (généralement il y a une application Camera).
  • Un bon gestionnaire de fichiers, tel que MiXplorer, pour examiner les fichiers et faire des copies de sauvegarde.
  • SQLiteViewer (ou toute autre application de gestion SQLite) peut être utile pour inspecter la base de données TopoDroid.


En outre, selon la façon dont vous traitez vos relevés topographiques, vous voudrez peut-être installer :

  • une visionneuse / éditeur d'images (généralement, il y a une visionneuse d'image préinstallée : Gallery)
  • un visualiseur / éditeur DXF (par exemple TurboViewer)
  • un visualiseur / éditeur de feuille de calcul (par exemple Google Sheets)
  • une visionneuse SVG (par exemple SVG viewer)
  • un archiveur ZIP (par exemple ZArchiver)
  • GoogleEarth
  • une visionneuse de fichiers de traces (par exemple, Androzic)

 

Proj4
TopoDroid stocke les coordonnées des points dans le système de référence géographique long.-lat. WGS-84. Les altitudes sont stockées à la fois par rapport à l'ellipsoïde et au géoïde.

L'application Proj4 est utilisée par TopoDroid de deux manières différentes pour transformer les coordonnées entre le WGS-84 et d'autres systèmes de coordonnées.

  • pour convertir les coordonnées d'un point du longitude (WGS-84) à un autre système de référence de coordonnées.
  • pour saisir les coordonnées d'un point dans un système de coordonnées différent.
