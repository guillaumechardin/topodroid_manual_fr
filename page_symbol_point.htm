OUTILS POINTS

Les outils "point" sont définis dans des fichiers texte.

Deux entrées spécifiques : "user" et "label" (texte), sont intégrées dans le programme. Les autres fichiers "points" sont téléchargés à partir du sous-répertoire symbol/point de TopoDroid. Ce répertoire est rempli avec les outils de points compilés dans l'apk de distribution. Vous pouvez supprimer les outils dont vous n'avez pas besoin, modifier ceux que vous souhaitez modifier et en ajouter de nouveaux qui ne sont pas fournis dans l'apk.


Le fichier d'outil de point contient des lignes spécifiant la propriété de l'outil de dessin :

  • "symbole point" [obligatoire]
  • "nom NAME", où NAME est le nom par défaut de la zone [obligatoire]
  • noms traduits, par exemple, "nom-es ..." [facultatif]
  • "th_name NAME", où NAME est le nom de type Therion et doit être le même que le nom de fichier Therion sans l'éventuel préfixe "u" [obligatoire]
  • "color rgb" (rgb au format hexadécimal) [obligatoire]
  • "orientation FLAG" [facultatif, faux si manquant, vrai si FLAG est '1' ou 'oui']
  • "has_text FLAG" [optionnel, faux si manquant, vrai si FLAG est '1' ou 'oui']
  • "roundtrip VALUE" [optionnel, pour SVG-roundtrip, VALUE peut être 1 (walls_shp), 2 (walls_sym), 3 (detail_shp), 4 (detail_sym) - défaut 4]
  • "level" view_flag [facultatif, niveau de visualisation du point ; si le niveau est absent, il est fixé à 1].
  • "path" commence le chemin d'accès du symbole de point. Ceci est spécifié avec les commandes suivantes :
    • "moveTo X Y"
    • "lineTo X Y"
    • "cubicTo X1 Y1 X2 Y2 X Y"
    • "addCircle X Y R": centre (X, Y), rayon R
    • "arcTo X0 Y0 X1 Y1 FROM_ANGLE SPAN_ANGLE": (X0, Y0 - X1, Y1) rectangle de délimitation, angles en degrés, 0 le long de l'axe X, direction est CW
  • "endpath" marque la fin du chemin d'accès de l'outil
  • "endsymbol"

Orientation
Si l'attribut "orientation" est défini, le symbole de point peut être pivoté.

Niveau
L'attribut "level" prend la valeur de l'indicateur des différentes vues activées sur les éléments de ce point.
L'indicateur est un nombre dont la somme est 1 (niveau de base), 2 (plancher), 4 (remplissages), 8 (plafond) et 16 (artefacts).
