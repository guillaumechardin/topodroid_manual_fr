PALETTE DE COULEURS PocketTopo

Quand cette boîte de dialogue est ouverte, sélectionnez le réglage de la palette de couleurs PocketTopo (dans la section Importer / Exporter).
Elle affiche un tableau avec trois colonnes :
  • Couleur PocketTopo
  • Ligne associée (nom de Therion)
  • Symbole de point associé (nom de Therion)


La palette de couleurs est utilisée dans l'importation des fichiers ".top" pour mapper les couleurs PocketTopo aux objets de dessin TopoDroid.
