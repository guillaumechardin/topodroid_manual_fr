SOUS-RÉPERTOIRES de TopoDroid

ATTENTION : les informations ci-dessous sur la façon dont les données du relevé topo, les données d'étalonnage et les croquis sont stockés en sous-répertoires dépendent de votre configuration Android. Ils peuvent se trouver soit sur votre mémoire interne (téléphone ou tablette) soit sur la carte SD additionnelle si vous en avez une voire répartis sur les deux. Explorez les mémoires de votre appareil avec un explorateur de fichiers pour vous en assurer.

L'application crée son répertoire de base par défaut, nommé "TopoDroid", dans la mémoire principale "Documents/TDX". Ce répertoire n'est pas supprimé lorsque vous désinstallez l'application. Dans ce répertoire, l'application crée la base de données SQLite "distox14.sqlite" qui stocke les données, les fichiers de croquis et les autres fichiers du relevé topographique.

L'application crée également des sous-répertoires pour les différents types de fichiers.
Les sous-répertoires suivants sont généralement créés dans le répertoire privé de l'application, c'est-à-dire "Android/data/com.topodroid.TDX/files" dans la mémoire externe primaire (généralement / sdcard/ ou ailleurs voir note ci-dessus).

  • bin : fichiers de firmware DistoX2
  • ccsv : fichiers texte CSV des étalonnages exportés
  • dump : fichiers de vidage de mémoire du DistoX
  • symbol : fichiers d'outils de dessin (points, lignes, zones)
  • man : traduction du manuel utilisateur

Le sous-répertoire "bin" est utilisé pour stocker les fichiers de firmware, à la fois les fichiers par défaut inclus dans TopoDroid et ceux issus du vidage de la mémoire récupérée à partir d'un DistoX2.
La base de données "device10.sqlite", contenant les données d'étalonnage et les informations sur les appareils, se trouve dans ce répertoire.

Pour chaque relevé topo, un dossier spécifique nommé d'après le nom de ce relevé est créé. Il contient les dossiers suivants :

  • audio : commentaires audio (fichiers .wav)
  • note : fichiers texte d'annotations du relevé topo
  • photo : images du relevé topo (fichiers .jpg)
  • tdr : fichiers de croquis binaires
  • out : fichiers exportés (cav, cave, csx, dat, dxf, png, svx, th, th2, top, tro).
  • c3d : exportation de croquis pour la représentation 3D (temporairement inutilisé).

Le sous-répertoire "zip" est utilisé pour stocker les archives compressées afin de les lire et de les décompresser dans TopoDroid. Lorsque vous souhaitez importer un relevé topo réalisé avec un autre programme (Therion, Compass, VisualTopo ou PocketTopo), vous devez le placer dans le sous-répertoire "import".

NOTA BENE IMPORTANT :

Dans les versions récentes de TopoDroid, les fichiers ci-dessous sont stockés et répartis dans les sous-répertoires de chaque relevé topographique inclus dans "Documents/TDX/Topodroid". Mais sur les versions plus anciennes ils étaient stockés dans "Documents/Topodroid" et regroupés dans un sous-répertoire spécifique à chaque type de fichier (contenant chacun les fichiers exportés de différents relevés).
En effet, les anciennes versions utilisaient le dossier de base "Topodroid" alors que les versions récentes utilisent le dossier de base "TDX". Ils ne doivent pas se mélanger et les deux version (< 5.1.40 et ultérieures) peuvent coexister.
Donc si vous avez utilisé une version plus ancienne, ces fichiers exportés précédemment sont restés stockés à l'ancien emplacement, pensez-y si vous les recherchez.

  • cave : fichiers Polygon
  • cav : fichiers Topo
  • csv : fichiers texte CSV des relevés topographiques exportés
  • csx : fichiers cSurvey (relevés topographiques exportées et croquis)
  • dat : Fichiers de boussole (relevés topographiques exportés)
  • dxf : fichiers DXF (relevés topographiques et croquis exportés)
  • grt : fichiers Grottolf (relevés topographiques exportés)
  • gtx : fichiers GHTopo (relevés topographiques exportés)
  • import : fichiers de relevé topo à importer (format Therion, Compass, VisualTopo et PocketTopo)
  • kml : GoogleEarth exporte [A]
  • plt : exportation de fichier trace OziExplorer [A]
  • png : fichiers image PNG (croquis exportés)
  • srv : fichiers Walls (relevés topographiques exportées)
  • svg : fichiers SVG (croquis exportés)
  • sur : Fichiers WinKarst (relevés topographiques exportés)
  • svx : fichiers Survex (relevés topographiques exportés)
  • Th : Fichiers Therion (relevés topographiques exportés)
  • th2 pour les fichiers calques de Therion (croquis sauvegardés)
  • th3 pour les fichiers de modèle 3D
  • tmp : pour les fichiers temporaires. Tout fichier restant dans ce répertoire peut être supprimé en toute sécurité.
  • top : fichiers PocketTopo (relevés topographiques exportés)
  • tro : fichiers VisualTopo (relevés topographiques exportés)
  • zip : archives ZIP des relevés topographiques (exportés et importés)
