ÉLÉMENT POINT DU CROQUIS

Cette boîte de dialogue s'ouvre lorsque vous sélectionnez un symbole de "Point" (en mode d'édition de la fenêtre de croquis) et que vous appuyez sur le bouton Propriétés de l'élément .

Propriétés du Point :

  • Taille : au choix XS, S, M, L, XL
  • Orientation : si le point est orientable
  • Texte : si le point possède un commentaire associé
  • Options supplémentaires Therion

 

Les vues du fond sont présentées sous forme de cases à cocher avec la lettre-code de la vue. Vous pouvez modifier les vues auxquelles le point appartient. L'icône du point n'est pas affichée si toutes ses vues sont désactivées. Le point sélectionnable sera néanmoins affiché en mode édition, de sorte que la boîte de dialogue d'édition du point reste accessible.

Annuler ferme la boîte de dialogue.
Enregistrer confirme les modifications et ferme la boîte de dialogue.

Lorsque le "glissement latéral" est activé, l'orientation du point peut également être modifiée en glissant vers le haut ou vers le bas et sur les côtés gauche ou droit du fond de page.


La touche RETOUR ferme la boîte de dialogue sans enregistrer les modifications.
