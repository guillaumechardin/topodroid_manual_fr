POINT DE SECTION TRANSVERSE DU CROQUIS

Cette boîte de dialogue s'ouvre lorsque vous sélectionnez un point de section et que vous appuyez sur le bouton Propriétés de l'élément dans la fenêtre de croquis (en mode "édition").

Propriétés du point :

  • taille (XS, S, M, L, XL)
  • couches, les vues du fond sur lesquels le point est affiché [T].
  • options Therion : le point de section du croquis a toujours l'option "-scrap" suivie du nom de la section transverse
  • x-section : case à cocher pour afficher le contour de la section transverse sur le croquis

Les vues du fond sont présentées sous forme de cases à cocher avec la lettre-code de la vue. Vous pouvez modifier les vues auxquelles le point appartient. L'icône du point n'est pas affichée si toutes ses vues sont désactivées. Le point sélectionnable sera néanmoins affiché en mode édition, de sorte que le dialogue d'édition du point reste accessible.


Annuler ferme la boîte de dialogue.
Enregistrer confirme les modifications et ferme la boîte de dialogue.


Le bouton Croquis permet d'ouvrir le croquis de la section transverse, si elle existe. Les modifications apportées au croquis de section transverse ne sont pas propagées au contour du croquis : pour le mettre à jour, fermez le contour et ouvrez-le à nouveau.


La touche BACK permet de fermer la boîte de dialogue sans enregistrer les modifications.
